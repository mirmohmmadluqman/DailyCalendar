Daily Calendar User Guide

Welcome to Daily Calendar

Daily Calendar is a versatile app designed to help you organize your schedule, manage notes, and stay productive. This guide will walk you through its features and how to use them effectively.

Core Features

Adding and Editing Notes

  • Title and Details: Enter a title (up to 50 characters) and detailed notes in the editor.
  • Color-Coding: Assign a color to your note for visual organization.
  • Categories/Tags: Add a category (e.g., Work, Personal) to classify your notes.
  • Priority Levels: Set a priority (Low, Medium, High) to highlight importance.
  • Recurring Notes: Make a note repeat daily, weekly, or monthly.
  • Event Duration: Specify start and end times to schedule events.
  • Attachments: Add URLs or text snippets to your notes (stored locally).
  • Pinning Notes: Pin important notes to the top of a day using the üìå Pin button.
  • Undo/Redo: Use the ‚Ü∫ Undo and ‚Üª Redo buttons while editing a note.
  • Voice Input: Click the üéôÔ∏è button to dictate your note title or body.
  • Auto-Save Drafts: Your note edits are automatically saved as you type.

Viewing and Managing Notes

  • Day Cells: Notes appear as titles in day cells, color-coded by priority and background.
  • Drag-and-Drop: Drag a note from one day to another to move it.
  • Sorting Notes: Notes are sorted with pinned notes at the top; others follow by entry order.
  • Holiday Display: Public holidays (e.g., New Year, Christmas) are highlighted in gold.

Searching Notes

  • Use the search bar to find notes by title, body, date, or category.
  • Click a search result to jump to that date and edit the note.
  • Clear the search using the ‚úñ button.
  • Filters: Search results automatically filter by matching criteria (e.g., date range, tags).

Bulk Editing

  • Hold Ctrl and click multiple days to select them.
  • Click the üìã Bulk Edit button to add a note to all selected days.

Exporting and Importing

  • Export: Click the Export button or press Ctrl + E to download your notes as a JSON file and events as an iCal file.
  • Import: Click the Import button or press Ctrl + I to upload a previously exported JSON file.

Clearing Notes

  • Click the Clear All button or press Ctrl + Shift + C to delete all notes (with confirmation).

Advanced Features

Reminders and Notifications

  • Notes with a start time will trigger browser notifications if permission is granted.
  • Notifications include the note title and body as a reminder.

Note Templates

  • Create reusable templates in the note editor (saved locally).
  • Select a template from the dropdown to prefill the note title and body.

Sharing Notes

  • Click the üîó Share button in the note editor to generate a shareable link (copied to clipboard).

Statistics Dashboard

  • Click the üìä button to view stats like total notes and most active days.

Weather Integration

  • Weather information for the current day is displayed (requires an API setup).

Customizations

  • Font Size: Adjust the app's font size in the Settings menu (‚öôÔ∏è).
  • Custom Themes: Choose from preset themes (Default, Blue, Green) in Settings.
  • Custom Background: Set a custom background color or image URL in Settings.
  • Multi-Language Support: Switch between languages (e.g., English, Spanish) in Settings.
  • High-Contrast Mode: Enable high-contrast mode for better accessibility in Settings.

Accessibility

  • Keyboard Navigation:
    • Ctrl + T: Jump to today.
    • Ctrl + E: Export notes.
    • Ctrl + I: Import notes.
    • Ctrl + Shift + C: Clear all notes.
    • Ctrl + N: Quick add note.
    • Tab: Navigate through interactive elements.
    • Enter or Space: Activate buttons or open day cells.
  • Screen Reader Support: All buttons and interactive elements have ARIA labels.
  • High-Contrast Mode: Enable in Settings for better visibility.

Tips and Tricks

  • Use color-coding and priorities to quickly identify important tasks.
  • Pin frequently accessed notes for easy reference.
  • Leverage recurring notes for daily reminders or routines.
  • Drag-and-drop notes to reschedule events effortlessly.
  • Check the statistics dashboard to track your productivity trends.
